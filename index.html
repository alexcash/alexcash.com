about the site.

The code and design for this site, aside from some common open source tools, were created entirely by me. In an effort to constantly grow and learn, I am always iterating on the design and code of this site. This is the third version of the site since launch just four months ago.

The web should be functional, aesthetically pleasing, and easy to understand, but to ensure compatibility, longevity, and ease of development there are some best practices that I always observe. I use the concepts of responsive web design, so that the user is always seeing content in a way that makes sense for their screen size. While I'm always pushing the envelope with new standards like CSS3 and HTML 5, I also strive to achieve graceful degradation, so that the site is functional and attractive even in legacy browsers.

My code is completely open source and available on Github. During development, I user descriptive variable and element names, comment my code, and make frequent version control commits with useful logs. When I'm ready for deployment, I concatenate my javascript, and I minify every single text based file, including my HTML, Javascript, and CSS so that my pages load blazingly fast.
